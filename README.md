## Project setup and Usage
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests (what tests?)
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
